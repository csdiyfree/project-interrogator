# Agent BE0 · v03 账号登录与鉴权

## 你的身份与任务

## 铁律

- 错误体沿用契约 `{"error":{"code","message"}}`;字段命名一致。

## 改动清单

### 1) 新建 `backend/app/auth.py`(写死用户 + 校验工具)
```python
# 写死的两个账号(无注册)。session_id 是登录成功后下发的不透明 bearer 令牌。
USERS = {
    "admin": {"password": "kaoda-2025-admin", "session_id": "sess-admin-9f3c1a2b8e7d"},
    "guest": {"password": "kaoda-2025-guest", "session_id": "sess-guest-4a6d2e9c1b07"},
}

VALID_SESSION_IDS = {u["session_id"] for u in USERS.values()}

def authenticate(username: str, password: str) -> str | None:
    """校验账号密码,成功返回该用户的 session_id,失败返回 None。"""
    user = USERS.get(username)
    if user is None or user["password"] != password:
        return None
    return user["session_id"]

def is_valid_session(session_id: str | None) -> bool:
    return bool(session_id) and session_id in VALID_SESSION_IDS
```

### 2) 新建 `backend/app/routers/auth.py`(登录端点,公开、无需鉴权)
```python
from fastapi import APIRouter, HTTPException
from pydantic import BaseModel
from app.auth import authenticate

router = APIRouter(prefix="/api", tags=["auth"])

class LoginRequest(BaseModel):
    username: str
    password: str

class LoginResponse(BaseModel):
    session_id: str

@router.post("/login", response_model=LoginResponse)
def login(body: LoginRequest) -> LoginResponse:
    session_id = authenticate(body.username, body.password)
    if session_id is None:
        raise HTTPException(
            status_code=401,
            detail={"code": "invalid_credentials", "message": "账号或密码错误"},
        )
    return LoginResponse(session_id=session_id)
```
在 `main.py` 里 `app.include_router(auth.router)`(与其他 router 一起挂载)。

### 3) 收紧 `require_session`(`main.py`)
把现有「有头即可」改为「必须是已知用户令牌」:
```python
from app.auth import is_valid_session

def require_session(x_session_id: str | None = Header(None, alias="X-Session-Id")) -> str:
    if not is_valid_session(x_session_id):
        raise HTTPException(
            status_code=401,
            detail={"code": "unauthorized", "message": "未登录或登录已失效"},
        )
    return x_session_id
```
其余端点不变——它们继续用 `require_session` 返回的 `session_id`(现在即用户令牌)做隔离,自动变为按用户隔离。

> 注意循环导入:`require_session` 现已依赖 `app.auth`;`app.auth` 不依赖 `app.main`,无环。`routers/auth.py` 不要 import `main`。
