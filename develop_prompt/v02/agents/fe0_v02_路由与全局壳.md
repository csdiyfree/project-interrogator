## 改动清单

### 1) 路由(`App.tsx`)
- 路由表改为:
  - `/` → `InfoPage`
  - `/upload` → `UploadPage`
  - `/resume/:resumeId` → `InfoPage`
  - `/interrogations/:interrogationId` → `InterrogationPage`(不变)
  - `/interrogations/:interrogationId/guide` → `GuidePage`(不变)
- **顶栏**:左侧品牌 `BrandMark` 链接到 `/`;右侧常驻两个图标按钮:
  - 「信息页」入口(列表/卡片状图标)→ `/`
  - 「上传」入口(加号 / 上传箭头图标)→ `/upload`
  - 沿用现有顶栏的玻璃拟态与配色,保持零文案。
- `InfoPage`/`UploadPage` 由 FE1 实现;**你先创建占位组件**(可仿现有 `pages/Placeholder.tsx`),让应用可编译运行。保留 `InterrogationPage`/`GuidePage` 引用不变。原 `ResumePage.tsx` 不再被路由引用(由 FE1 改造/删除)。

### 2) 类型(`api/types.ts`)
- `ResumeDetail` 增 `name: string | null`。
- `ProjectDetail` 增 `resume_id: string`。
- 新增:
  ```ts
  export interface ResumeListItem { id: string; name: string | null; status: ResumeStatus; created_at: string; }
  export interface ResumesList { resumes: ResumeListItem[]; }
  ```

### 3) endpoints(`api/endpoints.ts`)
新增两个函数(内部按 `USE_MOCK` 切换):
```ts
export function listResumes(): Promise<ResumesList> { if (USE_MOCK) return mock.listResumes(); return apiFetch('/api/resumes'); }
export function renameResume(resumeId: string, name: string): Promise<ResumeListItem> {
  if (USE_MOCK) return mock.renameResume(resumeId, name);
  return apiFetch(`/api/resumes/${resumeId}`, { method: 'PATCH', body: JSON.stringify({ name }) });
}
```

### 4) Mock(`mocks/handlers.ts`,必要时 `fixtures.ts`)
- `MResume` 增加 `name: string`、`created_at_iso: string`;`buildResume()` 生成名称(可用 `RESUME_SUMMARY` 里的姓名 + 当前日期,重名加 ` (n)`)。
- `getResume` 响应增加 `name`。
- `getProject` 响应增加 `resume_id`(`p.resume_id`)。
- 新增 `listResumes()`:返回**运行时创建的简历**(`buildResume` 产生的),按 created_at 倒序;状态用与 `getResume` 相同的 `parsing/parsed` 判定。**排除内置 seed-resume**,使「全新会话 = 空列表」可触发空状态;seed 实体仍可经 `seed-it-*` 深链直达(FE2/FE3 调试用)。
- 新增 `renameResume(id, name)`:更新 `MResume.name`,返回 `ResumeListItem`。
- 更新 `endpoints` 的 `import * as mock` 已涵盖新函数。

### 5) `lib/session.ts`
新增记忆上次打开简历的小工具:
```ts
const LAST = 'last_resume_id';
export function getLastResumeId(): string | null { return localStorage.getItem(LAST); }
export function setLastResumeId(id: string): void { localStorage.setItem(LAST, id); }
```
