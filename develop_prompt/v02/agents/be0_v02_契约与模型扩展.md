## 改动清单

### 1) `backend/app/models.py`
给 `Resume` 增加一列:
```python
name: Mapped[str | None] = mapped_column(String, nullable=True)
```
(放在 `status` 附近即可;可空,兼容历史行。)

### 2) `backend/app/schemas.py`
- `ResumeDetail` 增加字段:`name: str | None = None`。
- `ProjectDetail` 增加字段:`resume_id: str`。
- 新增:
```python
class ResumeListItem(APIModel):
    id: str
    name: str | None = None
    status: str
    created_at: datetime

class ResumesList(APIModel):
    resumes: list[ResumeListItem]

class ResumeRename(APIModel):
    name: str
```
> `ResumeListItem` 含 `created_at`,`APIModel` 已有 `created_at` 的 ISO 序列化器,直接复用。

### 3) 本地库迁移提示(写进改动说明/口头交付即可,不写迁移框架)
新增列不会被 `create_all` 自动加到既有表。交付时提醒:对 `backend/data/app.db` 执行 `ALTER TABLE resumes ADD COLUMN name VARCHAR;`,或删除该 db 文件让其重建(演示数据可弃)。
